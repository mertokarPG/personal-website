## Problem

Everyone wanted GenAI on the factory floor. Nobody had a safe, repeatable way to actually plug it into plant data without turning the LLM into a liability.

## Approach

1. Grounded the agent in Control Tower's read-only API — no direct PLC or MES access.
2. Scoped prompts to narrow, verifiable questions: 'What's line 3 running?', 'When does PO 12345 finish?'.
3. Hallucination budget: if the agent doesn't have a tool-call answer, it says so. No vibes-based factory advice.
4. Documented the pattern so sister sites could copy the integration, not the code.
