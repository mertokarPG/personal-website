## Problem

The ASRS ran 24/7 and had never had a safe path to test vendor software changes. Every update was a live-fire exercise. Operational losses were creeping up because no one wanted to touch anything.

## Approach

1. Inventoried every component: VMs, DBs, PLCs, HMIs. Cloned them into an isolated network segment.
2. Built a full mirror with 4 live PLCs, real firmware, synthetic load profiles.
3. Hardened the vendor connection path — new protocol, audit logging, least-privilege.
4. Every change now rehearses in emulator before prod. No exceptions, no 'small fixes'.
